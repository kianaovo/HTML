# 📝 在线备忘录

一个基于 Vue 3 的单页面在线备忘录应用，支持多用户注册/登录、备忘的增删改查、分类标签、搜索、导出以及深色模式。所有数据存储在浏览器本地，无需后端服务。

## ✨ 功能特性

- **🔐 多用户系统** — 支持注册与登录，使用 SHA-256 对密码进行哈希，session 持久化
- **📋 备忘 CRUD** — 创建、查看、编辑、删除备忘录，支持标题（最长 60 字）和内容（最长 500 字）
- **🏷️ 分类标签** — 工作、个人、紧急、学习、其他五类标签，支持按标签筛选
- **✅ 状态管理** — 进行中 / 已完成两种状态，可一键切换
- **🔍 搜索** — 支持按标题和内容实时搜索
- **📊 统计面板** — 展示全部、工作、个人、紧急、学习各分类的数量
- **⬇️ 导出功能** — 支持单条备忘导出和全部备忘导出为 `.txt` 文本文件
- **🌙 深色模式** — 一键切换浅色/深色主题
- **📱 响应式布局** — 适配桌面端与移动端，移动端支持侧边栏滑出
- **💡 交互细节** — Toast 通知、确认删除对话框、ESC 关闭弹窗、详情查看、新建用户示例数据

## 🚀 快速开始

1. 克隆仓库：

```bash
git clone https://github.com/kianaovo/HTML.git
cd HTML
```

2. 直接用浏览器打开 `index.html`，或使用任意静态服务器：

```bash
# 使用 Python
python -m http.server 8080

# 使用 Node.js (npx)
npx serve .
```

3. 打开浏览器访问 `http://localhost:8080`，注册账户即可开始使用。

## 🛠️ 技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | [Vue 3](https://vuejs.org/) (Composition API, CDN 引入) |
| 样式 | 原生 CSS (CSS Variables、Grid、Flexbox) |
| 数据存储 | `localStorage` (按用户隔离) |
| 密码哈希 | Web Crypto API (`SHA-256`) |
| 构建工具 | 无 — 纯静态 HTML，零依赖 |

## 📖 使用说明

### 注册与登录

首次使用需注册账号：用户名支持中文、英文、数字和下划线，密码至少 4 位。

### 创建备忘

点击右上角「新建」按钮或右下角「+」浮动按钮，填写标题、内容和分类标签后保存。

### 查看与编辑

- 点击备忘卡片查看详情
- 点击编辑图标修改备忘内容
- 点击勾选图标切换完成状态
- 点击删除图标移除备忘（需确认）

### 筛选与搜索

- 左侧边栏可按状态筛选：全部 / 进行中 / 已完成 / 紧急事项
- 顶部统计卡片可按分类筛选
- 搜索框支持按标题和内容实时过滤
- 排序按钮可切换「最新优先」/「最早优先」

### 导出备忘

- 单条导出：备忘卡片上的导出按钮
- 批量导出：工具栏「导出全部」按钮，将所有备忘导出为一个 `.txt` 文件
